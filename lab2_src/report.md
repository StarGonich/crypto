Проект состоит из следующих модулей:
1) `lsb.py` - основные функции для кодирования и декодирования сообщений
2) `crypto.py` - утилита с интерфейсом командной строки
3) `lsb_test.py` - модуль для тестирования и анализа
4) `utils.py` - вспомогательные функции для работы с изображениями

В качестве детектора выбран хи-квадрат.
